# 测试数据准备指南

## 1. 地球物理反演结果图像

### 图像要求
- **格式**: PNG, JPG, JPEG
- **内容**: 地球物理反演结果剖面图（如P波速度扰动、电导率等）
- **坐标信息**: 图像中应包含经度、深度等坐标轴标注
- **颜色标尺**: 应包含颜色标尺和数值范围

### 示例图像描述
如果您没有真实的地球物理图像，可以参考以下描述创建示例：

```
P波速度扰动剖面图 - 华北克拉通东部
- 横轴：经度 116°E - 118°E
- 纵轴：深度 0 - 50 km
- 颜色：蓝色表示低速异常（-4% 到 -2%），红色表示高速异常（+2% 到 +4%）
- 主要特征：
  - 117.2°E, 20-30km处：显著低速异常（深蓝色）
  - 116.5°E, 10-15km处：较小高速异常（红色）
```

## 2. 地理信息JSON文件

确保 `test_geo_info.json` 文件存在，格式如下：

```json
{
  "longitude_range": [116.0, 118.0],
  "latitude_range": [39.0, 41.0],
  "depth_range": [0, 50],
  "region_name": "华北克拉通东部"
}
```

## 3. 测试步骤

1. 准备反演结果图像并保存为 `inversion_result.png`
2. 确保地理信息JSON文件存在
3. 设置OPENAI_API_KEY环境变量
4. 运行测试脚本：

```bash
python test_input_parsing.py
```
